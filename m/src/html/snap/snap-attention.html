유의사항

본 파일은 <!DOCTYPE html> <html> ... </html> 까지 template 화 되어 s3 서버에 올라감 css 는 snap.css 와 동일 유의사항이 노출되는 단계를 살펴보면 1. 관리자가 snap admin 에서 유의사항 작성 후 등록 2. 본 파일의 article.snap-attention__article 안에 내용이 채워져 s3 서버에 저장됨 3. 사용자가 snap 등록화면에서 '유의사항' 클릭시 저장된 s3 경로를 노출시킴.
