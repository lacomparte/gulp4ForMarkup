POPUP OPEN WITH DIM

[사용법]
1. popup 을 오픈시킬 DOM 에 추가 할 properties
1-1. dim이 같이 열려야 하면 "data-dim='true'" 삽입
1-2. data-popup-open 에 "SOMETHING"

2. popup DOM 에 추가 할 property
2-1. data-popup="SOMETHING" 1-2의 "SOMETHING" 과 같은 값

POPUP OPEN ALONE

[사용법]
1. popup 을 오픈시킬 DOM 에 추가 할 properties
1-1. dim이 열리지 말아야 하면 "data-dim='false'" 삽입 또는 "data-dim" 미정의
1-2. data-popup-open 에 "SOMETHING"

2. popup DOM 에 추가 할 property
2-1. data-popup="SOMETHING" 1-2의 "SOMETHING" 과 같은 값

POPUP OPEN FULL

[사용법]
1. POPUP 을 오픈시킬 DOM 에 추가 할 properties
1-1. data-popup-open 에 "SOMETHING"

2. popup DOM 에 추가 할 property
2-1. data-popup="SOMETHING" 1-2의 "SOMETHING" 과 같은 값
2-2. data-popup-type 에 "full"

LAYER CLOSE

[사용법]
1. layer 를 닫을 DOM 에 추가 할 class
1-1. "data-layer-close='layerClose'" 추가


2. layer DOM 에 추가 할 class
2-1. class 에 'layer' 추가
